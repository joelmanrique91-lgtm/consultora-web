# Decisiones del proyecto (log)

Este documento registra decisiones técnicas y de producto para mantener trazabilidad.

## Decisiones

### 2026-01-09 — Estructura inicial del repositorio
- Se crea carpeta `docs/` para documentación.
- Se crea carpeta `site/` para el código del sitio web.
- Se pospone la elección del stack (framework/build) hasta definir contenidos y estructura.
  
### 2026-01-09 — Stack inicial del sitio
- Se adopta Astro para el sitio institucional (SSG por defecto, SEO y performance).
- Contenido inicial gestionado con Markdown.
- Estilos: CSS simple al inicio; se evaluará Tailwind más adelante si aporta velocidad sin complejidad excesiva.
